## 📈 Analytics (`/teacher/analytics`)

### Select a Course

- **Class Average** - Mean score
- **Highest Score** - Best performance
- **Lowest Score** - Lowest performance
- **Completion Rate** - % of students graded

### Charts
- **Grade Distribution** - Pie chart of grade ranges
- **Performance Trend** - Line chart over time

### At-Risk Students
- Students scoring below 50%
- Quick access to help them
- Email and current grade shown

## 💡 Tips & Tricks

### Dashboard
- Pending submissions count updates in real-time
- Click any stat card to go to that section
- Engagement chart shows weekly trends

### Courses
- Use search to quickly find a course
- Sort by students to see largest classes
- Click "Open" to view course details

### Submissions

- Search for specific students
- Click "Grade" to open grading interface

### Analytics

- Export report for records

## 🔧 Common Tasks

### Create a New Course
1. Click "Create Course" button
2. Fill in course details
3. Click "Continue Setup"

### Grade a Submission
1. Go to Submissions
2. Find the submission
3. Click "Grade"
4. Enter grade and feedback
5. Save

### View Course Analytics
1. Go to Analytics
2. Select a course
3. Review metrics and charts
4. Check at-risk students

### Find At-Risk Students
1. Go to Analytics
2. Select a course
3. Scroll to "At-Risk Students"
4. Click "Help" to assist

## 🆘 Troubleshooting

### Page Not Loading
- Refresh the page
- Check internet connection
- Clear browser cache
- Try different browser

### Data Not Showing
- Wait a moment for data to load
- Refresh the page
- Check if course is selected
- Verify you have courses

### Buttons Not Working
- Check if you're logged in
- Verify your role is teacher
- Try refreshing the page
- Clear browser cache
